Introduction to OCI SCIM

The following documentation provides the basic endpoints needed to utilize Oracle Cloud Infrastructure's SCIM.

Base Resource URLs
  • Base URL: https://<region-name>.scim.oci.oraclecloud.com/v2
  • Oauth Creds: https://auth.<region-name>.oraclecloud.com/oauth2/token
  • Core Schemas: https://<region-name>.scim.oci.oraclecloud.com/v2/Schemas
Parameters
  • grant_type: client_credentials
  • scope: urn:oracle:oci:scim-service
